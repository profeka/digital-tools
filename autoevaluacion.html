Web Final ZIP (20%)
0-4: No funciona, faltan páginas o CSS no vinculado.
5-8: Estética básica. Fallos menores en nombres o rutas.
9-10: Diseño pulido, Personalización de fuentes, rutas OK y navegación fluida.
Auditoría Oral (15%)

1. Estructura HTML

Localiza etiquetas h1, p, img sin dudar y explica para qué sirven.

2. Estilos CSS

Cambia color, tamaño o fondo usando el inspector de elementos o el código.

3. Rutas y Enlaces

Entiende rutas relativas, carpetas y soluciona fallos de carga de imágenes.

4. Validación IA

Explica código complejo generado por IA, detecta errores y borra lo innecesario.
